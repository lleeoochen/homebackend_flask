# Home Backend Server

### Frameworks
- Server: **Python Flask**
- Gateway Interface: **WSGI**
- Proxy Interface: **Nginx**

### Install package
- `./install package-name`

### Setup Server (Local)
- `./setup`

### Start Server (Local)
- `./start`
- Server will listen on localhost:8000

### Start Server (Remote)
- `sudo systemctl start/restart homebackend`
